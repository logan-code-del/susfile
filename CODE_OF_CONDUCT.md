# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.1. By participating, you agree
to respect other contributors and follow the rules described here.

Summary
- Be kind and respectful.
- No harassment, abuse, or hateful language.
- Report unacceptable behavior to the maintainers at `logan-code-del@example.com`.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
